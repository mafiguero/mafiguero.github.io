─── SMART HOUSE ───

Mi último gran proyecto y desafío ha sido diseñar y crear este portafolio. Esta ha sido mi primera página web y me ha servido para aprender HTML y CSS, ya que no tenía mucha experiencia con estos lenguajes.

A lo largo de mi formación en la informática he hecho y participado en numerosos proyectos, algunos de la universidad y otros de carácter personal. Es por ello que sentía la necesidad de crear este portafolio como repositorio de mis trabajos.









Para hostear este portfolio he utilizado una herramienta de Github que me permite visualizar mi proyecto de HTML y CSS con el dominio “github.io”.

Para desarrollar mi software y tener siempre en la nube mi proyecto actualizado he utilizado “git” junto con un repositorio personal de Github. De esta manera puedes tener un mejor control de versiones de tu software. Gracias a algunos proyectos de la universidad en los que ya había utilizado este tipo de técnicas de desarrollo software mediante uso de repositorios, no tuve problemas para incorporarlas a este proyecto.




Otra gran parte del diseño de esta web han sido sin duda los “assets”, para ello he hecho uso de algunos programas de edición de imágenes y de imagen vectorial.

A lo largo de mi vida he participado en muchos proyectos de diseño de logos o assets para comunidades, particulares o de caracter personal como es en este caso. Esto me ha servido para coger una buena habilidad en aplicaciones de edición de imagen como photoshop, gimp, adobe illustrator entre otras.

He querido cuidar cada mínimo detalle para conseguir un página web funcional, elegante, profesional y a la vez sencilla de implementar y programar, sin necesidad hacer uso de objetos de java script.

Como conclusión final quiero decir que me involucré en este proyecto para poder diseñar una página web personalizada que se adaptase perfectamente a mis necesidades. Así como aprender a programar en HTML y CSS y seguir formándome profesionalmente.
